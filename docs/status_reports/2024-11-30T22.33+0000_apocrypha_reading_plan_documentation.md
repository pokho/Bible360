# 365-Day Chronological Apocrypha Reading Plan

## Overview

This comprehensive 365-day reading plan covers all major apocryphal and deuterocanonical texts in chronological order, spanning from 300 BCE to 300 CE. The plan includes 41 unique texts organized into four historical phases with appropriate daily reading portions.

## URL Priority System

**Based on user preferences, the plan prioritizes:**
1. **BibleGateway NRSVUE** (First Priority) - For all deuterocanonical books
2. **Sacred-Texts.com** (Second Priority) - For all available texts
3. **EarlyChristianWritings.com** (Third Priority) - Backup when sacred-texts unavailable
4. **BibleGateway NABRE** (Fourth Priority) - Final backup for deuterocanonical books

## URL Access Implementation

The system automatically generates working URLs for each text based on the priority hierarchy:

### Primary URLs (First Priority)
- **BibleGateway NRSVUE**: `https://www.biblegateway.com/passage/?search=[Book]+[Chapter]&version=NRSVUE`

### Fallback URLs (Second/Third Priority)
- **Sacred-Texts.com**: Various patterns based on text category
- **EarlyChristianWritings.com**: `https://www.earlychristianwritings.com/[text].html`

### Source Descriptions
The system clearly indicates the source priority in the interface:
- "Source: Bible Gateway (NRSVUE - Preferred)"
- "Source: Sacred Texts (Preferred)"
- "Source: Early Christian Writings (Secondary)"
